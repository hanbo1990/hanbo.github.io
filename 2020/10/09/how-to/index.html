How to mount NFS

  • Step1: Install NFS sudo apt install nfs-kernel-server
  • Step2: Setup shared folder
    • sudo mkdir /var/nfs/share -p
    • sudo chown nobody:nogroup /var/nfs/share
    • add /var/nfs/share *(rw,nohide,insecure,no_subtree_check,async,no_root_squash) to /etc/exports
    • sudo service nfs-kernel-server restart
    • sudo mount -t nfs 127.0.0.1:/var/nfs/share /mnt to test it.
  • Mount NFS on linux board
    • mount -t nmount -t nfs -o nolock 192.168.31.23:/var /nfs/share /mnt/nfs, make sure nolock is used
