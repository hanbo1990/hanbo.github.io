ralink-wifi-setup

Posted on 2020-10-11 Edited on 2020-10-16 In linux Views: Valine:

This page records how to setup ralink usb wifi stick in a linux board.

generate ko files for the driver

  1. insert wifi usb into a ubuntu computer and do lsusb
  1. Go to linux wireless driver support to check if linux has driver for this device supported. Search 3070 and you can find it.

  1. Configure kernel configuration.

In kernel code drivers/net/wireless do grep "0x3070" * -nR

Make sure obj-$(CONFIG _RT2800USB)+= rt2800usb.o is in drivers/net/wireless/rt2x000/Makefile

search RT2800USB in drivers/net/wirless/rt2x00/Kconfig and you can find the device name in kernel config is Ralink rt27xx/rt28xx/rt30xx (USB) support.

do make menuconfig in kernel’s root folder and enable the following:

In Networking support/Wireless

  • -*- cfg80211 - wireless configuration API
  • -*- Generic IEEE 802.11 Networking Stack (mac80211)

In Library routines

  • {*} CRC-CCITT functions

In Device Drivers/Network device support/Wireless Lan

  • IEEE 802.11 for Host AP (Prism2/2.5/3 and WEP/TKIP/CCMP)
    • [*] Support downloading firmware images with Host AP driver
    • [*] Support for non-volatile firmware download
  • Ralink driver support
    • <M> Ralink rt27xx/rt28xx/rt30xx (USB) support

Save and make Image, make modules and copy all generated *.ko files into the board (together with the .bin file which is provided together with teh driver).

  • The bin file into /lib/firmware
  • The ko files into /lib/modules

Install drivers

  • insmod rt2x00lib.ko
  • insmod rt2x00usb.ko
  • insmod rt2800lib.ko
  • insmod rt2800usb.ko

Configure Wlan0

Configure wireless connection params in /etc/wpa_supplicant.conf:

Start Wlan0 by wpa_supplicant -B -Dwext -iwlan0 -c/etc/wpa_supplicant.conf -d up

Configure your Wlan0 ifconfig wlan0 up && ifconfig wlan0 192.168.xx.xx. Now try to ping a device to check if you are online!
